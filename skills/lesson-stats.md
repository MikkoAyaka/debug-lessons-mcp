# Lesson Stats / 踩坑统计

展示踩坑案例库的统计数据和趋势分析。

## 流程

1. 调用 `get_stats` MCP tool 获取全局统计。
2. 调用 `get_models` MCP tool 获取模型分布。
3. 结构化展示：
   - **总览**：案例总数、近 7 天/30 天新增
   - **按项目分布**：各项目案例数
   - **按分类分布**：高频踩坑类型识别
   - **按模型分布**：不同模型的案例数对比
4. 如发现明显趋势或异常（某分类激增），主动指出。

## 补充
- 想看特定项目：调用 `get_stats` 时传入 `project_id`
